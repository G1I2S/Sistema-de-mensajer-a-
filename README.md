# Sistema de Mensajería

Un sistema de mensajería moderno con interfaz de chat desarrollado con HTML y CSS puro.

## Descripción

Este proyecto presenta una interfaz de usuario para un sistema de mensajería que incluye una lista de contactos, estado de usuario y área principal para iniciar conversaciones.

## Características

- **Lista de Contactos**: Panel lateral con lista de contactos disponibles
- **Estado de Usuario**: Indicador de estado en tiempo real
- **Diseño Responsive**: Interfaz adaptable con flexbox
- **Botón Nuevo Chat**: Funcionalidad para iniciar nuevas conversaciones
- **Avatares Dinámicos**: Imágenes de perfil para cada contacto
- **Tema Oscuro**: Diseño moderno con fondo azul oscuro

## Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox)
- Fuentes: Arial, sans-serif

## Cómo Usar

1. Clona este repositorio:
   ```bash
   git clone https://github.com/G1I2S/Sistema-de-mensajer-a-.git
   ```

2. Abre el archivo `index.html` en tu navegador web preferido

3. Explora la interfaz del sistema de mensajería

## Visualización en Vivo
